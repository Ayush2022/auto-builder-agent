# Idea to Plan

## When to use this skill
Use this skill when the user asks to:
- build an app
- create a system
- design software
- develop a project

## Instructions

You MUST respond when triggered.

Analyze the user request and generate:

### 📦 Features
- List core features based on the idea

### 💻 Tech Stack
- Frontend
- Backend
- Database

### 🏗️ Architecture
- High-level system design

### 🚀 Steps
- Step-by-step development plan

Always provide a complete and structured answer.
